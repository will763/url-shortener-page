# url-shortener-page 💻

This is a project from Bootcamp Eduzz Fullstack Developer #3, for creating a link shortener api, in which I already developed the api in another repository, in this repository I developed the interface that consumes the api.

### Features 🪧

Users should be able to:

- View the optimal layout for the site depending on their device's screen size
- Shorten any valid URL
- Copy the shortened link to their clipboard in a single click
